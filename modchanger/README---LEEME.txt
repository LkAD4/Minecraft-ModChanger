----------ENGLISH---------

Welcome to the ModChanger program.  
This program is designed to automate the process of switching mods from one version to another when you have multiple versions of Minecraft.

---HOW IT WORKS---

In the same folder where this program is located, you can create a folder named after your Minecraft version to store your mods.  
The program allows you to create a folder to paste your mods into (option #1),  
or directly switch mods between versions (option #2).

--Steps--

Inside this folder, there is a document called "ruta" (path) where you must write the path to your Minecraft mods folder,  
which usually looks like this:  
`C:\Users\your user\AppData\Roaming\.minecraft\mods`

IMPORTANT: The path must be written exactly as shown in the example.  
Do not use quotes or parentheses.  
The program will not work without a valid path.

It’s a simple but very useful program.
